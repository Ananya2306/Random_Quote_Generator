# 💬 Quote Generator Web App

A clean and modern quote generator web application that allows users to view random inspirational quotes, switch between light/dark modes, and more. Built using HTML, CSS, and JavaScript with responsive design for both desktop and mobile devices.

---

## ✨ Features

- 🎲 Get random quotes with the click of a button
- 🌗 Toggle between light and dark modes
- 📱 Fully responsive and mobile-friendly
- 🎨 Smooth UI with transitions and gradients
- 🔁 No repeat quotes until all shown
- 🐦 Tweet this quote button
- 🖼️ Download quote as an image
- 🎨 Category switcher for filtering quotes

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Flexbox, Responsive Design)
- JavaScript (DOM manipulation)
- Google Fonts (`Poppins`)

---

## 🚀 How to Run This Project

1. Clone the repository:
   ```bash
   git clone https://github.com/Ananya2306/Random_Quote_Generator.git
   ```

2. Open the `index.html` file in any modern web browser.

> No backend or package installations required — this is a pure front-end project!
